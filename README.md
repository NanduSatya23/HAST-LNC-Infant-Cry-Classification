# HAST-LNC-Infant-Cry-Classification
HAST-LNC: Hybrid Audio Spectrogram Transformer Embedding with Lightweight Neural Classifier for Infant Cry Analysis
The HAST-LNC (Hybrid Audio Spectrogram Transformer Embedding with Lightweight Neural Classification) project focuses on automatically identifying the underlying reasons behind infant crying using deep learning–based audio analysis. Infant crying is the primary communication method for newborns, and understanding the reason for the cry is essential for caregivers and healthcare professionals. This project aims to develop an intelligent system that analyzes infant cry audio recordings and classifies them into different categories such as hunger, discomfort, tiredness, belly pain, and burping. The system utilizes real-world infant cry recordings from the Donate-a-Cry dataset, which contains user-uploaded audio samples collected in natural environments.

The proposed system combines pretrained Audio Spectrogram Transformer (AST) embeddings with lightweight machine learning classifiers to improve cry classification performance. The audio signals are first preprocessed through resampling, noise handling, and spectrogram conversion, after which transformer-based embeddings are extracted to capture temporal and spectral characteristics of infant cries. These embeddings are then used as input features for classifiers such as Logistic Regression and Multi-Layer Perceptron (MLP). While Logistic Regression provides a baseline linear classification model, the MLP classifier captures nonlinear relationships within the audio features, resulting in improved prediction accuracy. The proposed approach reduces the need for handcrafted acoustic features and enables efficient learning from high-level audio representations.

Overall, the project demonstrates how transformer-based audio representation learning combined with lightweight classification models can effectively detect infant cry patterns. The developed system can support smart parenting tools, automated baby monitoring systems, and healthcare-assistance applications by providing reliable and real-time insights into infant needs based on cry analysis.
