# Dataset Description

This project uses the **Donate-a-Cry Infant Cry Audio Corpus**, a publicly available dataset containing real-world infant cry recordings. The dataset consists of user-uploaded audio files that represent different infant needs such as hunger, discomfort, tiredness, belly pain, and burping.

The audio recordings are collected in natural environments and are not heavily processed, making them suitable for real-world cry analysis research. Each audio sample is stored in **WAV format** with durations typically ranging between **6–7 seconds**.

Dataset Statistics:

* Dataset Name: Donate-a-Cry Infant Cry Corpus
* Total Audio Samples: 457
* Audio Format: WAV
* Average Duration: 6–7 seconds
* Source: Kaggle

The dataset is used to train and evaluate the proposed **HAST-LNC (Hybrid Audio Spectrogram Transformer with Lightweight Neural Classifier)** framework for infant cry classification.
